Banking AI Chatbot

Duration: 6 months
Role: Lead Developer
Status: Completed

Overview

A comprehensive banking chatbot application with a user-friendly conversational interface for customers and an admin dashboard for internal management. Built to demonstrate core banking interactions in a conversational format, enabling users to perform banking operations through natural language commands while admins manage backend operations efficiently.

Key Features

  • Balance & Transactions: Check account balance and view recent transaction history
  • Fund Transfer: Transfer money between accounts seamlessly
  • Card Management: Request new cards, block/unblock existing cards
  • Deposits: Open Fixed Deposits (FD) and Recurring Deposits (RD) with maturity information
  • Loan Services: Apply for loans and track application status
  • Profile Management: Update personal and account details
  • Support System: Raise and track support tickets
  • NLP-Powered Intent Recognition: Advanced natural language processing for accurate understanding of user queries
  • Admin Dashboard: Manage users, review loan applications, handle support tickets, and view system statistics

Technologies Used

Flask Python MySQL HTML CSS JavaScript NLP RESTful API

Additional Libraries

sentence-transformers scikit-learn numpy spaCy NLTK

Architecture Highlights

Advanced NLP pipeline combining pattern-based intent recognition with machine learning models for natural language understanding. Modular architecture with separate routes for user authentication, admin operations, and chatbot message handling. RESTful API design with Flask backend connecting to MySQL database through optimized queries. Integration of sentence transformers and semantic similarity models for enhanced intent classification and entity extraction. API endpoints for third-party integrations and webhook support.

Challenges & Solutions

Challenge: Creating an intuitive conversational interface that accurately understands user intent without complex NLP infrastructure.

Solution: Implemented a lightweight keyword-based intent parser with pattern matching that covers common banking queries while maintaining simplicity and extensibility. Added optional ML models for users who want enhanced accuracy.

Results & Impact

  • Simplified banking operations through natural language interaction
  • Reduced administrative overhead with centralized dashboard management
  • Comprehensive audit logging for security and compliance
  • Modular design allows easy feature expansion and customization
