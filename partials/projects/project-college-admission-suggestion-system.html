College Admission Suggestion System

Duration: 6 months
Role: Lead Developer
Status: Completed

Overview

The College Admission Suggestion System is an ML-powered recommendation platform built for KEAM aspirants in Kerala. It predicts the most suitable colleges based on rank, course preferences, and geographic criteria. The system automates model selection, ensuring high prediction accuracy and offers dual interfaces: a student-facing web app and an admin desktop tool for dataset management and model training.

Key Features

  • Machine learning–based personalized college suggestions
  • Automatic best-model selection from multiple algorithms
  • Student-friendly responsive web interface built with Flask
  • Admin panel for database updates and ML model training
  • Real-time predictions in under 2 seconds
  • Support for multiple years of KEAM admission data (2017–present)

Technologies Used

Python 3.11 TensorFlow Flask HTML CSS SQLite Linear Regression Polynomial Regression Random Forest Support Vector Regression (SVR) Neural Networks

Additional Libraries

NumPy Pandas Scikit-learn Matplotlib (for admin visualization) Tkinter (admin GUI) Joblib (model persistence)

Architecture Highlights

Dual Interface System:Web interface for students and Desktop GUI for admins

Automatic ML Pipeline:Trains and evaluates models and Selects best based on accuracy and error metrics

Modular Code Structure: Separate modules for ML models, database operations, UI, and routing

Portable System: Runs locally without external dependencies

Challenges & Solutions

Challenge: Ensuring accurate predictions across diverse ranks and courses.
Handling multi-year KEAM admission variations.
Providing both administrative controls and student usage.
Maintaining performance and fast response.

Solution: Implemented model comparison and dynamic algorithm selection.
Used normalized datasets and preprocessing pipelines.
Developed separate interfaces: Tkinter for admin, Flask web app for users.
Optimized model loading using Joblib and indexed SQLite queries.

Results & Impact

  • Improved prediction accuracy due to model selection automation
  • Instant suggestions make the system usable for real-time decision support
  • Helps KEAM aspirants narrow down colleges based on real data, reducing uncertainty
  • Scalable and ready for deployment on cloud platforms
  • Demonstrates full-stack development, ML integration, and UX-friendly design
