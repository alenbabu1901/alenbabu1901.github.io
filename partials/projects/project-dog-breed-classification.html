Dog Breed Classification and Recommendation

Duration: 6 months
Role: Lead Developer
Status: Completed

Overview

The Dog Breed Classification and Recommendation System is an AI-powered platform that helps users identify dog breeds from images and receive personalized breed recommendations based on lifestyle and preference factors. The system combines deep learning–based image classification with a content-based recommendation engine and provides a responsive, user-friendly web interface.

Key Features

  • Image Classification using ResNet50V2 with transfer learning
  • Real-time prediction for 120+ breeds with high accuracy
  • Recommendation engine based on cosine similarity for personality and lifestyle matching
  • Flask web app with authentication, dashboard, and saved matches
  • Support for multiple image formats and secure file uploads

Technologies Used

TensorFlow Keras ResNet50V2 Cosine similarity NumPy Pandas Python Flask HTML5 CSS3 SQLite

Additional Libraries

OpenCV Scikit-learn Pillow

Architecture Highlights

Modular system architecture separating ML pipeline, recommendation logic, and web application

Saved TensorFlow model for fast inference without retraining

SQLite-backed profile storage for dog characteristics and user preferences

Flask API layer handles image classification, recommendation flow, and user session management

Challenges & Solutions

Challenge: High variation in image quality and resolution.
Improving accuracy across 120+ fine-grained breeds.
Mapping user preferences to breed traits.
Providing seamless user experience.

Solution: Applied data augmentation, normalization, and preprocessing pipeline.
Used transfer learning with ResNet50V2 and tuned epochs, dropout, and optimizers.
Implemented cosine similarity with multi-dimensional scoring and weight tuning.
Built a responsive interface with clear workflows and secure upload handling.

Results & Impact

  • Achieved high classification accuracy across 120+ breeds
  • Delivered meaningful and personalized breed suggestions based on lifestyle inputs
  • Built a complete end-to-end ML product: training → model deployment → user-facing interface
  • Demonstrates CNN deep learning, recommendation system design, full-stack development, and user experience thinking
